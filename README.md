Gems and Co is a basic e-commerce web application for a jewelry store, developed using **PHP, MySQL, HTML, CSS, and JavaScript**.  
The project demonstrates fundamental backend concepts such as user authentication, database connectivity, and cart management.

This project was built as a **learning project** to understand full-stack web development using PHP.

---

## Features
- User registration and login system
- Product listing for jewelry items
- Add to cart functionality
- View and manage cart
- Session-based authentication
- MySQL database integration

---

## Tech Stack
- **Backend:** PHP  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **Server:** Apache (XAMPP / WAMP / LAMP)
